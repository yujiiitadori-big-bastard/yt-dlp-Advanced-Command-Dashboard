yt-dlp Advanced Download Interface

Structured command reference for downloading video and audio using yt-dlp. Paste your own video URL where required.

Environment Setup

Install Python

  • Download the latest Python release from the official Python website.
  • Enable the option Add Python to PATH during installation.

Install yt-dlp

pip install -U yt-dlp

Install FFmpeg

  • Download the FFmpeg static build.
  • Add the bin directory to system PATH.

Windows 11 Setup Commands

pip install -U yt-dlp
winget install ffmpeg

Termux Android Setup

pkg update pkg upgrade pkg install python ffmpeg pip install -U yt-dlp

Download Commands

With Embedded Thumbnail

Full Quality Video + Metadata + Subtitles

  • Best video and audio streams
  • Embeds thumbnail, chapters and metadata
  • English and Hindi subtitles supported
  • Automatic retry and resume
py -m yt_dlp -f "bestvideo+bestaudio/best" -o "%(title)s.%(ext)s" --embed-thumbnail --embed-metadata --embed-chapters --embed-info-json --write-auto-subs --embed-subs --sub-langs "en.*,hi.*" --ignore-errors --retries infinite --fragment-retries infinite --continue VIDEO_URL
yt-dlp -f "bestvideo+bestaudio/best" -o "%(title)s.%(ext)s" --embed-thumbnail --embed-metadata --embed-chapters --embed-info-json --write-auto-subs --embed-subs --sub-langs "en.*,hi.*" --ignore-errors --retries infinite --fragment-retries infinite --continue VIDEO_URL

Extract Highest Quality MP3

  • Audio only extraction
  • MP3 conversion
  • Maximum VBR quality
  • Thumbnail embedded as album art
  • Metadata added automatically
py -m yt_dlp -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --add-metadata -o "%(title)s.%(ext)s" VIDEO_URL
yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-thumbnail --add-metadata -o "%(title)s.%(ext)s" VIDEO_URL

Download and Merge to MKV

  • Best video and audio streams
  • Merged into MKV container
  • Thumbnail converted and embedded
py -m yt_dlp -f "bv*+ba/b" --merge-output-format mkv --embed-thumbnail --convert-thumbnails jpg VIDEO_URL
yt-dlp -f "bv*+ba/b" --merge-output-format mkv --embed-thumbnail --convert-thumbnails jpg VIDEO_URL
