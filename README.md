# yt-dlp Advanced Command Dashboard

A clean GitHub Pages interface for high-quality yt-dlp commands.

## Features

- Full quality video download with metadata and subtitles
- High quality MP3 extraction with thumbnail embedding
- MKV merging workflow
- Copy-to-clipboard buttons
- No tracking
- No personal data
- Fully static

## Deployment

1. Create a new GitHub repository.
2. Upload `index.html`.
3. Go to Settings → Pages.
4. Select branch `main` and folder `/root`.
5. Save.

Your site will be live automatically.
